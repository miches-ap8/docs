## What is Unified API?

Unified API is the standardization of different vendor API data models and exposing them to you as 1 single service.

Bring your own solutions, bring your own business rules and let unified API handle the rest of the hard stuff. 

This integration allows you to hyperfocus on what really matters to you.  

* Problem with different connectors? Check.  
* Hysterical with field mappings from one service to another? Check. 
* Worried about vendor API updates? Check.
